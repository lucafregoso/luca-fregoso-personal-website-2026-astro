# Java — TDD Harness

- **Test runner**: JUnit 5 + Mockito; surefire/failsafe; aim for ~80% line coverage
- **RED first**: write the failing test before any production code.
- **GREEN minimum**: only the code that makes the test pass.
- **REFACTOR**: behaviour-preserving structural change; tests stay green.
